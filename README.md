# EduShare

Le logiciel EduShare, est un outil de travail numérique, permettant aux élèves comme aux professeurs de partager des fichiers entre eux.

Ce logiciel est sous licence [MIT]("https://github.com/likapi/edushare/blob/main/LICENSE") donc, toute personne, souhaitant réutiliser, le code source du logiciel y est autorisée, uniquement si elle n'oublie pas, de mentionner la fondation Likapi, en tant que propriétaire du code source.
